# Jelly Drift

Portrait-first jellyfish animation and mini-game for social media demos.

## Run

Open `/Users/abdullahmac/Documents/automation/index.html` in a browser, or serve the folder with a static server.

## Controls

- `Start Run` begins a 45-second round.
- Move the jellyfish with touch or mouse.
- Collect glowing blooms for score.
- Avoid spiky urchins or lose time.

## Notes

- The piece stays animated in demo mode, so it can be screen-recorded without interaction.
- Best score is stored in `localStorage`.
